Case study
Building Buildcloud

Background

Buildcloud is a kick ass technology start-up based in Leeds. They had a vision to create a cloud based project managment tool for architects, which we helped transform from a cool idea into a real product.

We worked with the Buildcloud founders to design and build an app that revolutionises how architects manage & communicate on projects.

Design

At its heart Buildcloud is about information & communication so the design needed to put this to the fore. The interface was kept clean, clear and simple with a minimal core colour pallet. Subtle animations where used to provide greater context when interacting with the user interface.

Technology

Buildcloud is built on the Laravel PHP framework. Users can interact using the web-based application or via an iOS mobile app, which uses the included REST API. Using Laravel allowed us to work with a range of server technologies. We also had flexibility should the server environments ever need to change. Furthermore, we could stick to best practices by working within the framework structure.

We also integrated Buildcloud with the Stripe payment provider and Quaderno VAT-MOSS service. This allowed users to receive a real-time tax breakdown before setting up their subscription. All without ever leaving their page within the app.
