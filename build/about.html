Our Process

Discovery & Research

The most important part of any new project. This stage sets the foundation for the rest os the project. We’ll ask a lot of questions about your project, produce documents such as a development specification, begin to develop user personas and set project goals. We also look at any data from existing sites and analyse your competition. Basicly we leave no stone unturned in getting to know you, your brand & your organisation.

Structure & Wireframe

Now we start the process of content layout and structure. This is where the user experiance design really begins. Wire frames allow us to start experimenting with user paths and content layout & hierarchy. We produce prototypes for testing to give us an idea of how the site will work and perform.

User interface Design

Next comes the really fun bit! We add a layer of gorgeous design. We sweat every detail from colour palet to illustrations to typography. Great design should facilitate the clear communication of the content not get in it’s way. The simplest solution is often the best.

Development

Development begins around the same time as wire framing and user interface design. The sooner we can get into code the better as it helps build fast, efficent reliable sites. HTML prototypes are made from the wire frames to idendity potential problems early. Once layout is finalised we begin to layer on the user interface design, interactions, animations and fine detail that create an awsome experiance for your users.
